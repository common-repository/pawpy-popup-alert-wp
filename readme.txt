Count the words and letters of WordPress articles, as well as calculate the time needed to read

=== Description ===
 This plugin to display your posts or title of your posts and your custom post
 type on you blog or website and you can set your favorite color to show a title of posts ...
Features include:
* It can be controlled by admin
* It can be controlled through the settings attached to this plugin
* Built-in WordPress locale codes
* It contains a file that supports the Arabic language
* Help bloggers know the number of words and letters in an article

= Installing manually: =

1. Unzip all files to the `wp-content/plugins/alert` directory
2. Log into WordPress admin and activate the 'PAWPY Popup Alert WP' plugin through the 'Plugins' menu
3. Go to *PAWPY Popup Alert WP > Home* in the left-hand menu to start it
